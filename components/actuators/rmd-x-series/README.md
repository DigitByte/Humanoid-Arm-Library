# RMD-X Series Actuator

Reference actuator family for this humanoid arm library.

## Intended Use
- Shoulder/elbow/wrist actuation for prototype arms.
- CAN-based closed-loop control with external host.

## Integration Checklist
1. Assign CAN IDs.
2. Validate power and current limits.
3. Confirm zero/reference before joint assembly.
4. Run basic motion test from software examples.
