# Repository Map

## Library Core
- `components/`: reusable component catalog (actuator, gearbox, joint, link, base).
- `assemblies/`: component compositions and part-to-component mapping.

## Mechanical
- `mechanical/cad/model/`: STL geometry for arm hardware.

## Software
- `software/python/code/`: motion and multi-actuator validation scripts.

## Documentation
- `docs/library/LIBRARY_GUIDE.md`: component authoring and library rules.
- `docs/overview.md`: legacy overview/reference document.
